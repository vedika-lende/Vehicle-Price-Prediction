# Vehicle Price Prediction using Linear Regression

## Overview

This project focuses on predicting vehicle prices using the Linear Regression Machine Learning algorithm.  
The model is trained using various vehicle-related independent parameters to estimate the selling price accurately.

---

## Objective

The main objective of this project is:

- To build a machine learning model for vehicle price prediction
- To understand regression analysis
- To perform data preprocessing and visualization
- To evaluate model performance

---

## Dataset Information

The dataset contains multiple attributes related to vehicles such as:

- Brand
- Year
- Fuel Type
- Transmission
- Mileage
- Engine Capacity
- Selling Price
